# Intelligent Solar Energy Forecasting

## Overview
This project forecasts solar AC power generation using weather parameters and time-series feature engineering.

## Features
- Weather-based regression modeling
- Lag feature engineering
- Random Forest Regressor
- MAE and RMSE evaluation
- Interactive Streamlit UI
- Hosted deployment

## Dataset
- Plant_1_Generation_Data.csv
- Plant_1_Weather_Sensor_Data.csv
- Sample dataset included in /data folder

## Model Performance
- MAE ≈ 15.36
- RMSE ≈ 52.52

## Deployment
Hosted on Streamlit Cloud.
